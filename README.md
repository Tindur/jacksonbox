# jacksonbox

Dependencies
--------------------------------------

1. [node](http://nodejs.org/download/)
2. [grunt](https://github.com/gruntjs/grunt-cli) npm install grunt-cli
2. [bower](http://bower.io/) npm install bower

Setup
--------------------------------------

Client

Might need elevation (sudo)

```bash
cd client
npm install
bower install
```

Server

```bash
cd server
npm install
bower install
```

Running
--------------------------------------

Client

```bash
cd client 
grunt serve
```

Server (for now)

```bash
cd server
node app.js
```
